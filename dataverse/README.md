# Dataverse Documentation

Dataverse is the system of record for the IT Helpdesk Agent.

The main table is `IT Ticket` / `IT Tickets`.

This table stores ticket state, requestor identity, escalation, notification, acknowledgement, SLA, AI summary, and audit fields.

See [it-ticket-table.md](it-ticket-table.md) for the grouped column list.

No Dataverse export in this repository should include tenant URLs, environment IDs, real user data, private ticket content, secret values, or connection details.
