# Agents

This folder holds public-safe Copilot Studio agent notes and sanitized exports for the IT Helpdesk Agent project. The Copilot Studio agent uses the display name `IT Support Assistant`.

The agent design is documented in [docs/project-summary.md](../docs/project-summary.md) (agent behaviour and user journeys) and [docs/architecture.md](../docs/architecture.md) (Copilot Studio layer). Agent evidence, including topics, authentication settings, and chat journeys, is in the [screenshot gallery](../screenshots/README.md).

Only sanitized, public-ready material belongs here — no raw exports, private working notes, tenant-specific settings, environment IDs, secrets, or screenshots that expose private data.
