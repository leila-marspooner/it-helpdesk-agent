# Agent Contributor Guidance

This repository documents a Power Platform / Copilot Studio portfolio project.

When updating this repo, use only public-safe documentation and sanitized project artifacts. Do not publish private local notes, raw working exports, tenant-specific details, or private user data.

Keep documentation professional, recruiter-friendly, and honest. Do not overclaim production readiness. Clearly distinguish features that are built, planned, demo-only, or work in progress.

Preserve project terminology, including:

- IT Support (Portfolio)
- ITHelpdeskPortfolio
- IT Technician Console
- Helper - Create Ticket
- Helper - Check Status
- Helper - Escalate Ticket
- Automation - Notify Manager
- TKT####
- RequestorAadObjectId
- CurrentUserAadObjectId
- AlreadyEscalated
- NotAuthorized
- NotFound
- InvalidFormat
- Success

Never include secrets, API keys, passwords, tokens, tenant URLs, personal emails, or private user data.

Explain changes clearly so the repository remains easy to review and maintain.
