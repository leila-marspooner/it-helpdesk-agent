# Repository Working Notes

This is a Power Platform / Copilot Studio portfolio project.

Use only public-safe documentation and sanitized project artifacts when updating this repository. Do not publish private local notes or raw working exports.

Keep documentation professional, recruiter-friendly, and honest.

Do not overclaim production readiness.

Clearly distinguish built, planned, demo-only, and work-in-progress features.

Preserve project terminology such as `IT Support (Portfolio)`, `ITHelpdeskPortfolio`, `IT Technician Console`, `Helper - Create Ticket`, `Helper - Check Status`, `Helper - Escalate Ticket`, `Automation - Notify Manager`, `TKT####`, `RequestorAadObjectId`, `CurrentUserAadObjectId`, `AlreadyEscalated`, `NotAuthorized`, `NotFound`, `InvalidFormat`, and `Success`.

Explain changes in plain English so the repository remains easy to review and maintain.
